# Custom Schemas

> For referring to tables on a schema other than `public`, the schema name should be prepended to the table (e.g. public.users, auth.providers or chat.messages)

## Adding a schema with Hasura

To add a schema, click the '+' symbol in the left menu in the Hasura console.

![Add new database schema in Hasura](/images/postgresql/new-schema.png)

You can change the schema you are in using the dropdown in the top left.

You can also create relationships between tables in different schemas.
