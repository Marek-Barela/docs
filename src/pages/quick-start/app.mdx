# App

Our backend is ready, let's build the frontend app!

## React app

Create a new react app:

```bash
npx create-react-app nhost-app
cd nhost-app
```

We'll start by adding a `jsconfig.json` to your React app to make imports easier. Create a file `jsconfig.json` in the root of your project:

`jsconfig.json`

```json
{
  "compilerOptions": {
    "baseUrl": "src"
  },
  "include": ["src"]
}
```

Create a new folder `src/components` and move `App.js` to the `src/components` folder.

Because we moved the `App.js` file you now must update the import of `App` in the `src/index.js` to: `import App from "components/App";`

Perfect, let's move on!

Install these packages that you'll use to connect your React App to your Nhost GraphQL API.

```bash
npm install react-nhost @apollo/client graphql graphql-tag
```

Start the app:

```bash
npm start
```

Your app should start as normal now.

## Add NhostApolloProvider to index.js

Wrapp `NhostApolloProvider` around your app so we can use GraphQL anywhere in the app.

> Replace `https://hasura-xxx.nhost.app/v1/graphql` with your project's GraphQL API URL. You can find your GraphQL API endpoint in the Dashboard in the Nhost console.

`src/index.js`

```jsx
import React from "react";
import ReactDOM from "react-dom";
import App from "App";
// import `NhostApolloProvider`
import { NhostApolloProvider } from "react-nhost";

// Wrapp `NhostApolloProvider` around `<App />`.
ReactDOM.render(
  <React.StrictMode>
    <NhostApolloProvider gql_endpoint="https://hasura-REPLACE.nhost.app/v1/graphql">
      <App />
    </NhostApolloProvider>
  </React.StrictMode>,
  document.getElementById("root")
);
```

## Fetch data using GraphQL in App.js

It's time to fetch data!

`src/components/App.js`

```jsx
import React from "react";
import { useQuery } from "@apollo/client";
import gql from "graphql-tag";

const GET_TODOS = gql`
  query {
    todos {
      id
      created_at
      name
      completed
    }
  }
`;

function App() {
  const { data, loading } = useQuery(GET_TODOS);

  if (loading) {
    return <div>Loading</div>;
  }

  return (
    <div>
      {!data ? (
        "No todos"
      ) : (
        <ul>
          {data.todos.map((todo) => {
            return <li key={todo.id}>{todo.name}</li>;
          })}
        </ul>
      )}
    </div>
  );
}

export default App;
```

That was easy. Your todos are now listed in your app!

## Add todos

We can now list todos, now let's add functionallity to our app so we can add todos.

We'll import `useMutation`, create a new GraphQL mutation and add a form that will add todos.

`src/compoentns/App.js`

```jsx
import React, { useState } from "react";
import { useQuery, useMutation } from "@apollo/client";
import gql from "graphql-tag";

const GET_TODOS = gql`
  query {
    todos {
      id
      created_at
      name
      completed
    }
  }
`;

const INSERT_TODO = gql`
  mutation($todo: todos_insert_input!) {
    insert_todos(objects: [$todo]) {
      affected_rows
    }
  }
`;

function App() {
  const { data, loading } = useQuery(GET_TODOS);
  const [insertTodo] = useMutation(INSERT_TODO);
  const [todoName, setTodoName] = useState("");

  if (loading) {
    return <div>Loading</div>;
  }

  return (
    <div>
      <div>
        <form
          onSubmit={async (e) => {
            e.preventDefault();

            try {
              await insertTodo({
                variables: {
                  todo: {
                    name: todoName,
                  },
                },
              });
            } catch (error) {
              alert("Error inserting todo");
              console.log(error);
            }
            alert("Todo inseted");
            setTodoName("");
          }}
        >
          <input
            type="text"
            placeholder="todo"
            value={todoName}
            onChange={(e) => setTodoName(e.target.value)}
          />
          <button>Add todo</button>
        </form>
      </div>
      {!data ? (
        "no data"
      ) : (
        <ul>
          {data.todos.map((todo) => {
            return <li key={todo.id}>{todo.name}</li>;
          })}
        </ul>
      )}
    </div>
  );
}

export default App;
```

You can now add new todos using the new form. That's great! But to see the newly added todos you need to reload the app. That's not great. Let's fix that using subscriptions!

## Get data in realtime using subscriptions

Subscriptions are used to get data in realtime from the database. When the data changes in the database we'll instantly have the new data in our app. It's almost like magic!

We'll change `useQuery` to `useSubscription` and we'll replace `query` to `subscription` in `GET_TODOS`.

`src/components/App.js`

```jsx
import React, { useState } from "react";
// replace `useQuery` with `useSubscription`
import { useSubscription, useMutation } from "@apollo/client";
import gql from "graphql-tag";

// replace `query` with `subscription`
const GET_TODOS = gql`
  subscription {
    todos {
      id
      created_at
      name
      completed
    }
  }
`;

const INSERT_TODO = gql`
  mutation($todo: todos_insert_input!) {
    insert_todos(objects: [$todo]) {
      affected_rows
    }
  }
`;

function App() {
  // replace `useQuery` with `useSubscription`
  const { data, loading } = useSubscription(GET_TODOS);
  const [insertTodo] = useMutation(INSERT_TODO);
  const [todoName, setTodoName] = useState("");

  if (loading) {
    return <div>Loading</div>;
  }

  return (
    <div>
      <div>
        <form
          onSubmit={async (e) => {
            e.preventDefault();

            try {
              await insertTodo({
                variables: {
                  todo: {
                    name: todoName,
                  },
                },
              });
            } catch (error) {
              alert("Error inserting todo");
              console.log(error);
            }
            alert("Todo inseted");
            setTodoName("");
          }}
        >
          <input
            type="text"
            placeholder="todo"
            value={todoName}
            onChange={(e) => setTodoName(e.target.value)}
          />
          <button>Add todo</button>
        </form>
      </div>
      {!data ? (
        "no data"
      ) : (
        <ul>
          {data.todos.map((todo) => {
            return <li key={todo.id}>{todo.name}</li>;
          })}
        </ul>
      )}
    </div>
  );
}

export default App;
```

Perfect! New todos gets listed when we add new todos. You can even open two tabs and add new todos in one tab and see how all windows displays the latest data.

Your app now have superpowers!
